How to run the code

Required Software:
	python v2.7, scikit-learn v0.13, nltk library
	
Instructions :
	1.Extract the content to same folder
	2.Run getwords.py using python getwords.py first to get the key words. Note that few files will be created.
	3.Run main.py using python main.py to get the result.txt file in your folder which will be your final file.
